Color Format Details

  • HEX: A hexadecimal color code starting with # followed by 6 characters (0-9, A-F)
  • RGB: Red, Green, Blue values each ranging from 0 to 255
  • HSL: Hue (0-360), Saturation (0-100%), Lightness (0-100%)
  • HSV: Hue (0-360), Saturation (0-100%), Value (0-100%)

How to Use Markdown Editor

Basic Formatting

  • # Heading 1
  • ## Heading 2
  • **Bold text**
  • *Italic text*
  • ~~Strikethrough~~

Lists

  • Unordered list:
    - Item 1
    - Item 2
  • Ordered list:
    1. First item
    2. Second item

Links & Images

  • Link: [Text](URL)
  • Image: ![Alt text](image URL)
  • Email: <address@example.com>

Code

  • Inline: `code`
  • Block: ```language
    code
    ```

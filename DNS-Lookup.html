How to Use This Tool

  1. Enter a domain name - Type the domain you want to check (e.g., example.com)
  2. Select record type - Choose the DNS record type you want to lookup
  3. Click "Lookup DNS" - The tool will query the DNS records and display results
  4. Review results - See all available DNS records for your query

Note: This tool queries public DNS servers and may not show private or local DNS records. Some domains may have restrictions on certain record types.

Common DNS Record Types

  • A Record - Maps a domain to an IPv4 address
  • MX Record - Specifies mail servers for the domain
  • CNAME - Creates an alias from one domain to another
  • TXT - Contains text information for various purposes
  • NS - Identifies authoritative name servers

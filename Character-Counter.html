How to Use This Character Counter

  1. Type or paste your text into the input box above

  3. Use the results to check length requirements for:

Common Character Limits

  • Twitter: 280 characters per tweet
  • Meta Title: 50-60 characters (optimal for SEO)
  • Meta Description: 150-160 characters
  • SMS Messages: 160 characters per message
  • Instagram Caption: 2,200 characters max

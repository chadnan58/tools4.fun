How to Use This Case Converter

  1. Enter your text in the input box
  2. Click on any case conversion button to transform your text
  3. View the converted text in the output box
  4. Copy the result to your clipboard with one click
  5. Use the clear button to start over

Case Conversion Types:

  • UPPERCASE: Converts all letters to uppercase
  • lowercase: Converts all letters to lowercase
  • Sentence case: Capitalizes the first letter of each sentence
  • Title Case: Capitalizes the First Letter of Each Word
  • camelCase: removes spaces and capitalizes each word except the first
  • PascalCase: Removes Spaces And Capitalizes Each Word
  • snake_case: replaces_spaces_with_underscores
  • kebab-case: replaces-spaces-with-hyphens

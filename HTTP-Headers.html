Common HTTP Headers Explained

  • Content-Type: Specifies the media type of the resource
  • Server: Information about the software used by the server
  • Cache-Control: Directives for caching mechanisms
  • Set-Cookie: Sends cookies from the server to the user agent
  • Content-Security-Policy: Prevents cross-site scripting attacks
  • Strict-Transport-Security: Enforces secure connections to the server

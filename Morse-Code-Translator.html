How to Use This Morse Code Translator

  1. Enter text in the left box to convert to Morse code
  2. Click "Text → Morse" to translate text to Morse code
  3. Enter Morse code in the right box (using . and - with spaces between letters and / between words)
  4. Click "Morse → Text" to translate Morse code back to text
  5. Use the "Play" button to hear your Morse code

Note: Letters are separated by spaces, words by forward slash (/).
